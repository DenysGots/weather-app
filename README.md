# Weather App
Simple weather widget, desktop and mobile layout

Deployed on Heroku:
http://app-simple-weather.herokuapp.com/

Note: used AccuWeather API has a limit of 50 calls/day due to Limited Trial

Start local server:
* clone
* run `npm i`
* run `dev:local`
* open browser at `http://localhost:5400`
